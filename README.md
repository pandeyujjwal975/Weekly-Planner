# Weekly Planner

Weekly Planner is a simple productivity web application built using HTML, CSS, and JavaScript.  
It helps users plan and manage tasks for each day of the week in an organized way.

The app stores tasks using browser localStorage, so data remains saved even after refreshing the page.

---
Live at : https://pandeyujjwal975.github.io/Weekly-Planner/

## Features

- Plan tasks for all 7 days of the week  
- Add and delete tasks easily  
- Data saved automatically using localStorage  
- Mobile and desktop friendly design  
- No external libraries or frameworks used  

---

## How to Use

1. Open the app in your browser  
2. Choose a day of the week  
3. Enter a task and click **Add**  
4. The task will appear under that day  
5. Tasks remain saved even after page reload  

---

## Technologies Used

- HTML – Structure  
- CSS – Styling and layout  
- JavaScript – Logic and localStorage handling
